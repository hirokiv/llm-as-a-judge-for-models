# 冪等性保証

!!! info "開発中"
    このページは実装フェーズの進行に合わせて更新されます。

## 概要

冪等性とは、同じ入力に対して常に同じ出力を返す性質です。

## LLMにおける冪等性の課題

LLMは確率的なモデルであり、完全な冪等性の保証は困難です。

## 本システムのアプローチ

### 1. モデル・バージョン固定

- 特定のモデルバージョンを使用
- 例: `gpt-4-0613`

### 2. パラメータ固定

```python
{
  "temperature": 0,
  "seed": 42,
  "top_p": 1.0
}
```

### 3. 冪等性検証

同じ入力で複数回実行し、一致度（variance_score）を計算：

```
variance_score = (一致回数) / (総実行回数)
```

## 目標値

- **variance_score >= 0.9**: 冪等性が高い
- **variance_score < 0.9**: 冪等性に問題あり
